# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2025-11-23

### Fixed

- Fixed an issue with PreSend method

## [1.0.2] - 2025-11-23

### Changed

- Renamed `traefikCompatibilityPreSend` to improve clarity

## [1.0.0] - 2025-11-22

### Added

- Initial release
- Komga API node with authentication support (username/password and APIKey)
- Docker compose setup for local testing with Komga and n8n
- Support for Books endpoints (get, list)
- Support for Collections endpoints (create, delete, get, list, update)
- Support for Libraries endpoints (create, delete, get, list, update)
- Support for Readlists endpoints (create, delete, get, list, update)
- Support for Series endpoints (get, list)
